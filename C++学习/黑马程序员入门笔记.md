C++的两种模版机制**函数模版**和**类模版**。	

#### 1.2.1函数模版语法

建立一个通用函数，其函数返回值类型和形参类型可以不具体指定，用一个虚拟的类型来代表。

```cpp
template<typename T>
函数声明或定义
```

示例

```cpp
template<typename T>

void swap(T &a, T &b)
{
  T temp;
  T = a;
  a = b;
  b = T;
  return ;
}
//不需要重复定义不同类型的参数
//类型参数化

int a = 1;
int b = 2;
swap(a,b);     //	自动类型推导
swap<int>(a,b);   //制定类型
```

#### 1.2.2函数模版注意事项

- 自动类型推导，必须推导出一致的T。
- 模版必须要确定T的数据类型，才可以使用。

```cpp
template<typename T>
void func()
{
  cout<<"error";
  
}
	func();  // 错误，没有确定T的类型


```
